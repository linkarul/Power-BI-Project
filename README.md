# Power-BI-Project
My Sample Power BI Project with the financial data provided in the excel "Financial Sample.xls" is given below. Here various visualizations like Donut chart, Scatter Chart, Stacked Column Chart, Stacked Bar Chart,Line and Stacked column chart and KPI cards to display KPI values are used.

It depicts various measures like product and its units sold details, product sales by country, month, and manufacturing cost against the sales price along with minimum and maximum values of units sold.
